What is this website about?

This website is a collection of experimental pages created to showcase my knowledge over html/css.


I was working in this webpage while learning the basics of css through the brocode html/css course.

What to expect?

Expect a really simple project using just the basics of css such as:

  • Text and Font manipulation.
  • Using Borders for Debugging.
  • Dealing with Shadows.
  • Building layouts with Float and Flexbox.
  • Working with Containers and Boxes.
  • Exploring Dimensions, Displays and Positions.
  • Applying basic interactivity with Pseudo-classes.
  • Creating simple Navigation Bars and a Dropdown menu.
  • An Image Gallery-like page.
  • Importing CSS Icons from Third-Party sites.
  • Basic Animations.
